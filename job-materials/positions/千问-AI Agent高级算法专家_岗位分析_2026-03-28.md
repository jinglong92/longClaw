## JD要点（基于截图人工提取）
### 关键词
- Agent
- 大模型
- 搜索
- 强化学习

### 职位定位
- 主导通用AI Agent（对标 Manus 等前沿形态）的核心算法研发与落地。
- 面向大模型驱动的信息搜集、智能决策、智能办公等场景，构建“搜索增强 + 自然对话”能力。

### 具体职责（两张截图汇总）
1. **算法研发与创新**：
   - 牵头通用 AI Agent 核心算法设计
   - 覆盖大模型微调与强化学习、智能规划与决策、多模态信息融合、搜索增强机制
   - 持续提升 Agent 通用性与任务执行能力
2. **场景落地与优化**：
   - 聚焦搜索、对话等核心场景
   - 推动算法方案落地实施，结合业务需求迭代模型效果
   - 解决上下文理解、意图识别、多轮交互连贯性等问题
3. **团队管理与赋能**：
   - 带领 5-10 人算法团队
   - 制定研发计划、统筹项目进度、搭建高效协作机制
   - 指导团队成员成长，提升整体技术水平与创新能力
4. **技术攻坚与前瞻布局**：
   - 跟踪国内外通用 Agent/大模型前沿（论文与行业实践）
   - 主导技术预研与攻关，推动成果转化
5. **跨部门协作与国际化支持**：
   - 与产品、业务深度协同，明确落地路径
   - 涉及海外业务时，推进海外场景算法适配与技术支持
6. **Agentic-RL 与持续学习（新增截图）**：
   - 构建基于环境反馈的强化学习体系（RLHB/RLAIF）
   - 在模拟环境进行大规模探索，提高 Agent 在极端场景的鲁棒性与决策上限
   - 引入高频领域（学习/办公/生活）知识的长程预训练与持续学习
7. **数据驱动的评测与迭代（新增截图）**：
   - 建立系统化 Agent 评测体系：线上数据分析 + Bad Case 挖掘
   - 重点攻坚幻觉、误回答、组合泛化、执行失败等问题
   - 迭代手段不局限于线上微调，强调世界模型模拟、合成数据与 AI 反馈强化学习
8. **统一认知架构与 Skill Library（第4张截图）**：
   - 对接阿里内部及外部第三方服务，构建高效稳定的 Agent 框架
   - 设计 skill-based「统一Agent」架构，强化长程规划与自我反思能力
   - 协调推理/记忆/执行模块，提升跨场景、长链任务表现
   - 推动动态 skill library：能力可学习、可组合、可演化，并探索自动化技能发现与组合优化
9. **长链任务优化（第4张截图）**：
   - 在复杂任务中做任务分解 + 多步推理优化
   - 聚焦降低长链执行中的错误累积与漂移

### 任职要求（截图补全）
1. **学历背景**：计算机/AI/数学相关专业硕士及以上。
2. **工作经验**：5年以上 AI/大模型算法研发经验；有通用 Agent、智能对话、搜索推荐等核心研发经历优先。
3. **技术能力**：
   - 精通大模型原理与应用（如 Transformer、预训练与微调）
   - 熟练 Python 与 TensorFlow/PyTorch
   - 具备搜索算法、NLP、智能决策等扎实功底
4. **项目经验**：
   - 主导复杂 AI 系统（如对话机器人、智能搜索引擎、Agent 产品）
   - 具备从算法设计到落地的完整经验，能独立解决高难问题
5. **架构与系统化能力（新增截图）**：
   - 有从零构建 Skill-based 系统或大型 Multi-Agent 协作系统落地经验
6. **综合素质（新增截图）**：
   - 逻辑与沟通协同能力强、主人翁意识强、适应快节奏业务

### 加分项
1. 顶会论文（NeurIPS/ICML/ICLR/ACL/EMNLP 等）
2. 3年以上算法团队管理经验（搭建与培养团队、复杂项目统筹）
3. 参与开源大模型或通用 Agent 相关项目并有社区贡献

## 与你的匹配度（快速）
- 匹配度：**高（A-）**
- 原因：
  - 与你“LLM/Agent + OR/决策”方向高度同频，尤其是“规划/决策 + RL + 搜索增强”。
  - 明确要求团队管理（5-10人），与你当前“补管理叙事”目标一致。
  - 业务是高杠杆平台型岗位，适合放大你系统级交付优势。

## 风险点（面试可能追问）
1. **管理门槛更高**：JD加分项写明“3年以上算法团队管理经验”，需给出可量化带队成果（效率、稳定性、业务指标）。
2. **研究前沿要求**：有“技术攻坚+前瞻布局”职责，可能追问论文阅读/复现/技术选型依据。
3. **Agentic-RL 实战门槛**：新增截图点名 RLHB/RLAIF、模拟环境探索、持续学习，可能深挖训练稳定性与 reward 设计。
4. **统一Agent架构门槛**：新截图强调 skill-based 统一认知架构 + 动态 skill library，面试中可能追问模块边界、可插拔设计与跨场景迁移能力。
5. **Agent工程化**：从 Demo 到生产化的架构、成本、延迟、可靠性治理。
6. **国际化适配**：若涉及海外业务，可能追问跨区域数据分布差异和策略迁移方案。

## 面试准备建议（针对该JD）
1. 准备一个“搜索增强 + 决策规划 + 执行反馈”的闭环案例（含指标）。
2. 准备一个“带5-10人团队”的管理案例页：机制、冲突、结果。
3. 准备一个“从模型能力到业务ROI”的算账页：效果增益、成本变化、上线节奏。
4. 若可行，补一页 Agent 评测框架：任务成功率、幻觉率、工具调用正确率、时延/成本。
