## 岗位来源

- 标题：蚂蚁集团-企业经管智能算法专家
- 薪资：50-80K
- 地点：杭州·西湖区·西溪
- 经验/学历：3-5年 / 硕士

## JD要点（基于截图人工提取）
### 我们提供的
1. 聚焦企业数智化核心战场：财务（预算/成本）与经营分析高价值场景，算法直接影响企业管理效率与资源分配。
2. 端到端主导权：从数据融合、建模、上线到效果归因可全流程掌控，并有机会定义“智能决策”标准。
3. 技术组合：时序预测 × 因果推断 × 运筹优化 × 可解释AI，多学科融合为日常工作形态。
4. 强协同环境：业务-算法-工程铁三角，强调落地与复杂问题求解。

### 你需要做的
1. 将时序预测、因果推断、运筹优化等能力转化为可落地算法产品，实现“数据洞察 -> 智能决策”闭环。
2. 主导端到端算法 pipeline：数据融合 -> 特征工程 -> 模型训练 -> 在线服务 -> 效果评估 -> 持续迭代。
3. 探索 AI 与经管知识融合新范式：规则约束下的预测校准、多目标资源分配、人机协同决策界面。

### 任职要求（来自第2张截图）
1. 精通 Python 与主流 ML 库（sklearn / PyTorch / LightGBM）。
2. 至少具备一类实战经验：
   - 时序预测：在招募量、人力成本、财务指标等场景应用 LSTM / Transformer / TFT / Prophet；
   - 智能优化：使用 OR-Tools / Gurobi / PuLP 解决排期、预算分配、资源调度；
   - 因果与可解释：在业务归因、策略评估中应用 SHAP / Causal Forest / PSM。
3. 有模型服务化、A/B 实验、监控告警等工程落地经验，能独立推进项目上线。
4. 计算机/统计/运筹/工业工程相关背景；一线大厂经验优先。
5. 加分项：主导预测/优化项目并产生显著业务价值；在 KDD / WWW / INFORMS 等顶会有论文。

## 与你的主线匹配度（快速）
- 匹配：高
- 原因：
  - 与你“AI + OR 融合（调度优化、运筹求解、供需预测）”主线高度一致。
  - 明确强调因果+优化+预测的组合，而非单一建模，符合你的差异化竞争方向。
  - 端到端与闭环导向利于你建立“业务价值可量化”的职业护城河。

## 风险点（先验）
1. 若你的经历更偏平台/模型研究，需补足“财务预算/经营分析”业务语境与指标叙事。
2. “企业经管”场景常涉及复杂组织协作，跨团队推进能力是隐性门槛。
3. 若仅展示算法精度而缺少ROI（降本、提效、预算偏差收敛等），竞争力会明显下降。

## 下一步建议（可执行）
1. 准备 2-3 个“预测+因果+优化联动”的闭环案例，每个案例写清业务指标提升与归因方法。
2. 单独补一页“经管场景方法论”：预算编制、成本归因、资源分配、在线策略与离线评估一致性。
3. 简历关键词对齐：`经营分析/预算成本/时序预测/因果推断/运筹优化/可解释AI/端到端落地`。
