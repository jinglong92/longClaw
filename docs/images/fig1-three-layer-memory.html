三层记忆架构总览

L1 上下文窗口 · 当前 Session
内容：最近对话 / 工具调用结果 / 当前轮决策
访问：直接在 context，无需检索
⏱️ 生命周期：Session 关闭即消失
round > 20 → 压缩
话题结束 → 归档
L2 每日日志 · YYYY-MM-DD.md
内容：对话流水 / 决策记录 / 实体更新
访问：memory_search.py Level 2/3
📅 生命周期：天 ~ 周，手动清理
key_conclusions → 提炼
L3 长期记忆 · MEMORY.md 分域块
内容：稳定偏好 / 重要结论 / 长期关注
访问：启动时按域注入 + 检索
📆 生命周期：月 ~ 年，主动维护
📋 JSONL 索引 · memory_entries.jsonl
内容：每条记忆的结构化元数据（域 / 实体 / 重要性 / 日期）    访问：memory_search.py 的底层数据源    来源：L3 → memory_entry.py 构建索引
